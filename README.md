dnsmess
=======
An IPv4 DNS lookup/reverse lookup tool with full request/response output.  The
DNS server user is hardcoded to Google Public DNS (see `dnsmess.h`).

Usage
-----
    $ make
    $ dnsmess [DOMAIN | IP]
